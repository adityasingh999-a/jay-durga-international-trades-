# 📖 Overview

Jay Durga Trade International is an international trading and distribution company specializing in premium cosmetics, skincare, beauty, and personal care products.

This project represents a modern, luxury-focused digital presence designed to:

* Showcase international cosmetic brands
* Build trust with distributors and retailers
* Generate partnership opportunities
* Strengthen global brand recognition
* Deliver a premium user experience

The website combines elegant design, responsive layouts, smooth animations, and modern web technologies to create a high-end corporate experience.

# ⚙ Installation

## Prerequisites

* Node.js 18+
* npm

Verify installation:

```bash
node -v
npm -v
```

---

## Clone Repository

```bash
git clone https://github.com/yourusername/jay-durga-trade-international.git
```

```bash
cd jay-durga-trade-international
```

---

## Install Dependencies

```bash
npm install
```

---

## Environment Setup

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---
